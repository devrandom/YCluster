VM Usage

GPU-hours per VM. Billable: lifecycle intervals marked billable (scheduler/web, or CLI with --bill). Tracked: all lifecycle intervals. Observed: periodic samples of actual incus state. A red Δ (observed − tracked) means runtime nobody's events accounted for — manual incus ops, host reboot autostart, or missed events. Now: whether the current run is billable (untracked = running per samples with no lifecycle event).
