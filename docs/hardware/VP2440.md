# Protectli VP2440

The Protectli VP2440 is a compact fanless firewall appliance repurposed as a storage node. It has 4x Intel ethernet ports (2x 10GbE SFP+, 2x 1GbE) in a small form factor.

## Specs

| Component | Detail |
|-----------|--------|
| CPU | Intel N150, 4C/4T, 700-3600 MHz (Alder Lake-N) |
| RAM | 16 GB DDR5-5600 (single DIMM, running at 4800 MT/s) |
| Storage | Samsung SSD 860 EVO M.2 250GB (SATA), 29GB eMMC (BJTD4R) |
| GPU | Intel Alder Lake-N integrated (UHD Graphics) |
| BIOS | AMI, v1.00, 2025-07-24 |

The 250GB SSD is a placeholder; it will be replaced with a larger drive (2TB+) before production.

## Network Interfaces

| Interface | PCI ID | Controller | Speed | MAC prefix |
|-----------|--------|------------|-------|------------|
| `enp1s0f0np0` | 01:00.0 | Intel X710 [8086:1572] | 10GbE SFP+ | 64:62:66 |
| `enp1s0f1np1` | 01:00.1 | Intel X710 [8086:1572] | 10GbE SFP+ | 64:62:66 |
| `enp2s0` | 02:00.0 | Intel I226-V [8086:125c] | 2.5GbE | 64:62:66 |
| `enp3s0` | 03:00.0 | Intel I226-V [8086:125c] | 2.5GbE | 64:62:66 |

### Interface Mapping vs MS01 (s1-s3)

The VP2440 has the same NIC chipsets as the MS01 but at different PCI addresses, resulting in different interface names:

| Role | MS01 (s1-s3) | VP2440 (s4) |
|------|-------------|-------------|
| Cluster (10GbE SFP+) | `enp2s0f0np0` | `enp1s0f0np0` |
| Uplink (1GbE) | `enp87s0` | `enp3s0` |
| AMT (1GbE) | `enp89s0` | `enp2s0` |

SFP+ transceivers are required for the X710 ports (not included).

## Disk Layout (Current)

Installed with Ubuntu defaults (not PXE):

| Partition | Size | Type | Mount |
|-----------|------|------|-------|
| sda1 | 1 GB | vfat (EFI) | /boot/efi |
| sda2 | 2 GB | ext4 | /boot |
| sda3 | 229.8 GB | LVM (`ubuntu-vg`) | |
| ubuntu-lv | 100 GB | ext4 | / |
| (free) | 129.8 GB | unallocated in VG | |

The 29 GB eMMC (`mmcblk0`) is unused.

**Note**: Storage playbooks expect VG name `vg0` with a `ceph` LV. When the disk is replaced, partition as:

| Partition | Size | Notes |
|-----------|------|-------|
| EFI | 512 MB | vfat |
| /boot | 4 GB | ext4 |
| LVM (vg0) | remainder | |
| vg0/root | 300 GB | ext4 |
| vg0/ceph | remainder (~1.7 TB for 2TB disk) | for MicroCeph |

## Differences from MS01

- **No Intel AMT** — The VP2440 does not have Intel vPro/AMT. No remote KVM or SOL available. Management requires physical access or SSH.
- **Different NIC names** — Same chipsets but different PCI topology. Requires `host_vars/s4.yml` for Ansible.
- **Fanless** — Passively cooled. Monitor thermals under Ceph load.
- **No WiFi** — MS01 has MT7922 WiFi; VP2440 does not.
